Requêtes sur la BDD immobilier

Afficher le nom des agences : SELECT * FROM agence

1bis en le triant par nom : SELECT * FROM agence ORDER BY nom ASC

2/ Afficher le numéro de l'identifiant de l'agence Orpi : SELECT idAgence FROM agence WHERE nom = 'orpi'

3/ Quel est le premier enregistrement de la table "logement" ? : SELECT * FROM logement LIMIT 0,1

Si on veut les deux premiers : SELECT * FROM logement LIMIT 0,2

3bis/ Quel est le dernièr enregistrement de la table logement ? SELECT * FROM logement WHERE idLogement = (SELECT max(idLogement) FROM logement)

4/ Affichez le nombre de logement (avec un alias "nombre_de_logement) ? : SELECT COUNT(*) AS 'Le nombre de logements' FROM logement

5/ Afficher les logements à vendre à moins de 150 000 € dans l'ordre croissant des prix :
SELECT * FROM logement
WHERE prix < 150000
AND categorie = 'vente'
ORDER BY prix

6/ Le nombre de logement(s) en location (alias 'nombre') : SELECT COUNT(idLogement) AS 'nombre' FROM logement WHERE categorie = 'location'

7/ Afficher les différentes villes recherchées par les personnes demandeuses d'un logement : SELECT DISTINCT ville FROM demande

8/ Le nombre de bien(s) à vendre par ville :
SELECT ville, COUNT(ville) AS 'nombre de biens en vente'
FROM demande WHERE categorie ='vente'
GROUP BY ville

9/ Quels sont les logements destinés à la location ?
SELECT idLogement
FROM logement
WHERE categorie = 'location'

10/ Quels sont les ids des logements entre 20 et 30m² ?
SELECT idLogement FROM logement WHERE superficie BETWEEN 20 AND 30

11/ Quel est le prix vendeur (hors commission) du logement le moins chère à vendre :
SELECT min(prix) AS 'prix minimum' FROM logement WHERE categorie = 'vente'

12/ Dans quelles villes se trouvent les maisons à vendre ?
SELECT ville from logement WHERE genre = 'maison' AND categorie = 'vente'
